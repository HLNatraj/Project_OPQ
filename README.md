# Project_OPQ

## Overview

This project is designed to help students explore and join various **courses** and **bootcamps** online. The platform provides a user-friendly interface for students to browse available courses, register for bootcamps, and raise queries regarding course details.

## Features

- **Course Listing**: View all available courses with detailed information including syllabus, duration, fees, and instructors.
- **Bootcamp Registration**: Join upcoming bootcamps with just a few clicks.
- **Online Registration**: Students can register for any course or bootcamp online through a simple and secure form.
- **Query System**: A built-in query submission feature where students can raise questions or seek clarification about any course or bootcamp.
- **Responsive UI**: Accessible across all devices including desktops, tablets, and mobile phones.

## Target Users

- Students looking to upskill or learn new technologies.
- Individuals interested in short-term or intensive training programs like bootcamps.
